# Example Output Files

This directory contains example output files generated by the data pipeline.

## Sample Files

After running the pipeline, you will see files like:

1. **bls_data_20231026_120000.json** - Raw BLS API response
2. **bls_processed_20231026_120100.csv** - Processed employment data
3. **bls_analysis_20231026_120200.json** - Statistical analysis

## Sample Data Structure

### Raw JSON (bls_data_*.json)
```json
{
  "status": "REQUEST_SUCCEEDED",
  "responseTime": 156,
  "message": [],
  "Results": {
    "series": [{
      "seriesID": "CES0000000001",
      "data": [
        {
          "year": "2023",
          "period": "M09",
          "periodName": "September",
          "value": "157151",
          "footnotes": [{"code": "P", "text": "Preliminary"}]
        }
      ]
    }]
  }
}
```

### Processed CSV (bls_processed_*.csv)
```csv
year,period,periodName,value,footnotes
2023,M09,September,157151,Preliminary
2023,M08,August,157267,
2023,M07,July,157116,
```

### Analysis JSON (bls_analysis_*.json)
```json
{
  "total_records": 48,
  "date_range": {
    "start": "2020-M01",
    "end": "2023-M12"
  },
  "statistics": {
    "mean_value": 152345.67,
    "median_value": 153200.0,
    "min_value": 142250.0,
    "max_value": 157267.0,
    "std_dev": 4321.23
  },
  "yearly_average": {
    "2020": 142540.0,
    "2021": 148120.5,

    "2023": 156200.8
  }
}
```

## Understanding the Data

- **Values**: Employment in thousands (e.g., 157151 = 157,151,000 employees)
- **Period Codes**: M01-M12 represent months January-December
- **Footnotes**: P = Preliminary data
- **Series**: CES0000000001 tracks total nonfarm employment in the US

## Data Refresh

Files are timestamped to allow tracking multiple runs over time. The pipeline always processes the most recent files.
